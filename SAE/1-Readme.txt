Bienvenue dans le readme de l'application "zabchat" conçu dans le cadre de la SAE R302 et R309

dans ce repertoire, vous trouverez 3 répertoires :
- Client
- Docummentation
- Server

Comme leurs noms l'indique ces 3 répertoires contiennent chacun des fichiers spécifiques a leurs roles
 
====== Server ======
 ce repertoire contient :
- un fichier .bat utile lors de l'installation du serveur sur un systeme d'exploitation Windows
- le fichier .py est le fichier a exécuter afin de démarer le serveur (mot de passe : root)
- le fichier zabchat.sql qui contient la base de donnée (utilisateur de la base de données : zabchat | mot de passe : sdfghjkl)
 
 
====== Client ======
 
 ce repertoire contient :
 - un fichier .bat utile lors de l'installation du client sur un systeme d'exploitation Windows
 - un fichier .py qui correspond au programme a executer
 
====== Documentation ======
 
 ce repertoire contient toute la documentation du projet dans son intégralité : installation détaillée, description des fonctions,...
